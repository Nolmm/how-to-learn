# __Проектная работа по верстке__ 
Одностраничный сайт с описанием методик обучения, статистическими данными и полезными обучающими ресурсами. 
Сайт содержит образовательные видео. 
Для создание проекта использовались технологии:
1. Flex.
2. Iframe.
3. Анимации и трансформации.

## __Планы по доработке проекта__
1. Создать форму обратной связи.
2. Переделать сетки с помощью технологии Grid.
